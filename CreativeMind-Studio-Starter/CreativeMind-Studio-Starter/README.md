# CreativeMind Studio

AI-powered creative strategy room and decision intelligence engine.

## Core Flow
Raw Idea → AI Strategy War Room → Audience Simulation → Creative Scorecard → Stress-Test Pivot → Refined Idea → Production Package → Final Report

## Stack
Frontend: React, Tailwind CSS, React Router, Recharts, Framer Motion  
Backend: FastAPI, Pydantic, PostgreSQL/Supabase  
AI: IBM Granite with LangGraph or native orchestration

## IBM Requirement
IBM Granite must power at least one real product workflow. IBM Bob may assist development, but should not be the only IBM tool used.
